# Changelog

All notable changes to Inline SQL are documented here.

## 0.1.1

- Add configurable SQL detection, dialect, and formatter settings.
- Improve multiline SQL formatting and diagnostics.
- Update examples and disclosure metadata for marketplace submission.

## 0.1.0

- Highlight SQL embedded in strings using settings-driven regular expressions.
- Support multiple matching patterns per host language.
- Add PostgreSQL-first dialect-aware highlighting and popular SQL dialects.
- Highlight aliases, alias references, and positional or named placeholders.
- Format embedded SQL with Sqruff, SQLFluff, or custom stdin formatters.
- Preserve host-language formatting by participating as a secondary formatter.
